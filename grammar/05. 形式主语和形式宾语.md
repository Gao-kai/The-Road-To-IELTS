# 形式主语和形式宾语

## 形式主语

在英语中，如果一个句子的主语部分是一个动词不定式或者动名词短语构成时，往往会导致主语部分过长，出现头重尾轻的情况。
举个例子，一般情况下当我们需要表达`做什么事对某人来说是什么什么的`或者`做什么事令某人感受怎么怎么样`的时候，
我们通常会采用主系表结构来写句子，例如：

```bash
To learn English well is easy.
Doing homework every day is difficult for me.
```

可以看到上面着两句话的主语部分都是动词不定式或者动名词短语，此时我们就要使用形式主语it来代替，然后将真正的主语放在句子末尾.

```bash
It is easy to learn English well.
It is difficult for me to do homework every day.
```

### 核心句型

记住，it做形式主语的句型常见的就下面这一种主系表结构：

```bash
it(形式主语) is +  adj(形容词)/n.(名词) + （for sb） +  to do / doing / 从句
对于某人做某事是怎样的感受
```

绝大多数情况下使用动词不定式to do构成的短语来充当真正的主语。
当表语是下面这些形容词的时候，一般会使用动名词doing来充当真正的主语.

- no use 无用的
- no good 不好的
- no help 没有帮助的
- nice 好的
- enjoyable 享受的
- worthwhile 值得的
- useless 无用的

```bash
It is no help for me playing computer games too long.
It is enjoyable playing basketball after work.
```

## 形式宾语

当动词不定式（短语）、动名词短语、从句在句子中充当宾语的成分的时候，为保持句子结构平衡，我们通常会使用形式宾语it来代替真正的宾语。
而将真正的宾语放在句子末尾。

### 核心句型：主谓+it+宾语补足语(多数是形容词)+to do / doing / 从句

一旦句子中主语的谓语动词是如下这些的时候，我们就要将it作为形式宾语。

- find 我发现
- think 我认为
- make 我使
- feel 我感到
- consider 我考虑
- suppose 我认为

```bash
I find it good to read books every day.
I think it difficult to understand his words.
He make it a rule never to borrow money from others.
She feel it pleasant that she can chat with AI.
I find it uneasy to insist on learning English every day.
```

### 核心句型：主谓+it+when/if/that从句

当需要表达某件事情发生的时候主语的感受是喜欢或者厌恶的时候，发生的这件事情用形式宾语it来代替，真正的宾语放在句子末尾。
一般情况下当谓语动词是表达喜好的动词时：

- like
- love
- dislike
- enjoy
- hate
- prefer
- appreciate

```bash
I like it when my team wins a game.
I prefer it if i did not have to do so much homework.
I hate it when people talk with their mouths full.
She dislike it if you arrive late.
```

### 高级句型：主谓语（不及物动词）+ 介词 + it（形式宾语）+ that从句

当谓语动作是不及物动词的时候，后面不能直接加宾语，需要跟介词来连接，
但当介词后面直接跟that从句（真正的宾语），必须加上it充当形式宾语。

```bash
You may depend on it that we shall always help you.
```
