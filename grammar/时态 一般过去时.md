# 一般过去时

一般过去时表示动作发生在过去，并且没有一直持续到现在，这一点要和过去将来完成时区分开来。

## 使用场景

1. 表达说话者在说话之前发生的动作或者状态
2. 表达过去的习惯或者过去经常发生的动作

## 一般过去时的标志词

1. yesterday系列
   - yesterday morning
   - yesterday afternoon
   - yesterday evening
   - yesterday night
2. last系列（上一个、上次、上次的）
   - last week
   - last month
   - last year
   - last time
3. ago系列（之前）
   - two days ago
   - three months ago
   - one week ago
4. in系列（in后面必须加过去的年份）
   - in 1998
   - in 2008.
5. on系列（on后面必须加过去的时间）
   - on Sunday
   - on this week

## 动词的一般过去式规则变化

1. 一般情况，动词词尾直接加ed
   - play → played（玩）
   - work → worked（工作）
   - watch → watched（观看）

2. 以不发音的字母e结尾的单词只需要加d
   - live → lived（居住）
   - like → liked（喜欢）
   - hope → hoped（希望）

3. 以辅音字母+y结尾，需要先变音为i，再加ed
   - study → studied（学习）
   - carry → carried（搬运）
   - try → tried（尝试）

4. 重读闭音节（辅音字母结尾并且重读），如果末尾只有一个辅音字母需要双写辅音字母再加ed（除了r，x，y结尾的单词）
   - stop → stopped（停止）
   - drop → dropped（掉落）
   - prefer → preferred（更喜欢）

## 动词的一般过去式不规则变化

1. 与动词原型一致
2. 动词过去式是-ought或者aught 无规律

   - think → thought（思考）
   - fight → fought（战斗）
   - buy → bought（购买）
   - bring → brought（带来）
   - catch → caught（捕捉）
   - teach → taught（教学）
  
3. 动词以eep/eel/ell/ill结尾 无规律

   - keep → kept（保持）
   - sleep → slept（睡觉）
   - feel → felt（感觉）
   - sweep → swept（扫）
   - tell → told（告诉）
   - sell → sold（销售）
   - smell → smelt（闻）
   - spill → spilled（溢出）

4. 动词以eed、ead、eet这种双元音字母结尾的时候 以ed或et结尾过去式
   - lead → led（阅读）
   - feed → fed（喂食）
   - meet → met（遇见）

5. 以end或者ild结尾，要将d替换为t
   - spend → spent（花费）
   - send → sent（发送）
   - lend → lent（借）
   - build → built（建造）

## 含be动词的一般过去时的句型转换

1. 肯定句：主语+be动词的过去式（was/were）+其他
   - I was born in 1998.
2. 否定句: 主语+be动词的过去式（was/were）+ not + 其他
   - He was not a teacher last year.
3. 疑问句：Be动词的过去式（was/were）+主语+其他
   - Was she a teacher last year?

## 含实义动词的一般过去时的句型转换

1. 肯定句：主语+动义动词的过去式+其他
   - I played basketball in NBA last year.
2. 否定句: 主语+ 助动词(do/does/did)+ not + 动义动词原型+ 其他
   - He did not work in Google last year.
3. 疑问句：助动词(do/does/did)+主语+ 动词原型+ 其他
   - Did he work in Google last year?
