# Git Commit Message Rules

1. **Language**: Must be in English.
2. **Length**: Keep it short, typically one sentence.
3. **Structure**: Use the format: "`<Verb>` `<what was changed>`"
4. **Examples**:
   - "Add new vocabulary examples"
   - "Fix grammar explanation in chapter 2"
   - "Update reading notes for lesson 3"
5. **Best Practices**:
   - Start with a capital letter
   - Use present tense verbs
   - Be specific but concise
   - Avoid punctuation at the end
